## Environment notes
- Backend: `./mvnw spring-boot:run` on port 8080. `OPENROUTER_API_KEY` must be in env.
- Frontend: `ng serve` on port 4200.
- SSE streaming sometimes delivers words without inter-word spaces in the raw `textContent()`. Regex assertions on LLM output must handle both spaced and non-spaced text (e.g. `niemog[ęe]` and `nie mog[ęe]` both need to match).
- Angular Material expansion panels (like "Podsumowanie zgłoszenia") must be clicked open before asserting on their inner `<dd>` elements — they are in the DOM but hidden until expanded.
- `getByText('X')` without `exact:true` does substring matching and can cause strict-mode violations when the same word appears in both a summary `<dd>` and an LLM reply `<p>`. Always use `exact: true` or `.first()` for summary field assertions.
